# Podcast App


## Overview

A React podcast browsing application.

Users can:

- Search podcasts
- Filter by genre
- Sort podcasts
- Browse pages
- View podcast details


## Features


### Search

Live title searching.


### Filtering

Genre filtering using provided genre data.


### Sorting

Options:

- Newest
- Title A-Z
- Title Z-A


### Pagination

Shows podcasts in pages of 12 items.


## Setup


Clone repository


Install:

npm install


Run:

npm run dev
